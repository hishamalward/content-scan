# Content Pool

The living backlog of things worth writing about. Append a row whenever something fascinates you (only `idea` is required; fill the rest later). The /content-scan skill appends rows it finds in your repos with a `content-scan YYYY-MM-DD` source tag.

Statuses: `seed` (raw) → `drafted` (full text exists) → `posted` (live, add date) → `retired` (decided against; keep the row, note why).

Pillars: define 3-4 themes you actually want to be known for, plus one rotational "everything else" pillar with a hard cap (about 1 in 5) so the feed stays focused.
